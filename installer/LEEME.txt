Advance Control - Instalador local

Contenido de esta carpeta:
- AdvanceControl.appinstaller
- AdvanceControl-x64.msix
- AdvanceControl-signing.cer
- Install-LocalInstaller.cmd
- Install-LocalInstaller.ps1
- Trust-LocalInstallerCertificate.cmd
- Trust-LocalInstallerCertificate.ps1

Uso recomendado:
1. Si quieres instalar localmente sin depender de internet ni de GitHub:
   - Ejecuta Install-LocalInstaller.cmd

2. Si quieres instalar de forma autoactualizable desde GitHub Releases:
   - Ejecuta Install-AutoUpdateFromGitHub.cmd
   - Este flujo solo funciona si la URL remota es accesible desde la PC destino
   - Si el repositorio es privado, GitHub puede bloquear la descarga del AppInstaller/MSIX

Si solo necesitas confiar el certificado manualmente:
- Ejecuta Trust-LocalInstallerCertificate.cmd

Si PowerShell esta restringido, usa los archivos .cmd en lugar de los .ps1.
