Advance Control - Instalador GitHub

Contenido de esta carpeta:
- AdvanceControl-signing.cer
- Install-AutoUpdateFromGitHub.cmd
- Install-AutoUpdateFromGitHub.ps1
- Trust-LocalInstallerCertificate.cmd
- Trust-LocalInstallerCertificate.ps1

Uso recomendado:
1. Ejecuta Install-AutoUpdateFromGitHub.cmd
2. Acepta la elevacion UAC cuando Windows la solicite
3. El script confia el certificado local y descarga el AppInstaller desde GitHub Releases

Este flujo solo funciona si:
- la PC destino puede acceder a GitHub
- existe un release reciente con AdvanceControl.appinstaller y AdvanceControl-x64.msix

Si solo necesitas confiar el certificado manualmente:
- Ejecuta Trust-LocalInstallerCertificate.cmd

Si PowerShell esta restringido, usa los archivos .cmd en lugar de los .ps1.
